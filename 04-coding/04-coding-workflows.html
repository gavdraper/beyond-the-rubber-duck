Coding Workflows

Efficient Development Patterns with AI

⚡ Break Prompts Down
Break prompts down into the smallest tasks you can, then make them smaller
📋 Persist Plans
Ask for a plan to be persisted to a markdown file with checkboxes
🔍 Be Critical of Plans
Be very critical of the plan
🎯 Be Specific
Be very specific
🌐 Provide Maximum Context
Give as much context as possible, specify files where you can, the AI can go and search a project but its context window is limited so it will just sample and miss context
