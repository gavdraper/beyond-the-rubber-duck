What Gets Stored

Everything in your conversation becomes part of the context

Included in Context:

  • Your messages and AI responses
  • File contents that were read
  • Code snippets and outputs
  • Error messages and logs
  • Search results and tool outputs
  • Custom instruction files

Compression

When context fills up, older information gets summarized or removed

Compression Process:

  • Oldest messages summarized first
  • Repetitive content removed
  • Key decisions and context preserved
  • Recent activity prioritized
  • File contents may be truncated

When to Clear

Recognize the signs that it's time for a fresh start

Clear When You Notice:

  • AI forgetting earlier decisions
  • Inconsistent responses
  • Switching to a new project/task
  • Performance degradation
  • Conflicting instructions building up

Presenter Notes

  • Context Limits: Claude has ~200k tokens, GPT-4 varies by model
  • Analogy: Like clearing your desk - sometimes you need a fresh workspace
  • Performance: Longer context = slower responses and higher costs
  • Strategy: Use files for persistent info, context for current work
  • Tools: Claude Code `/clear` command, new chat in web interfaces
