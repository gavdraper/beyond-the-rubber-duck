Key Recommendations:

  • Build out your custom instruction files:
    • In project (source control)
    • Personal home directory (non-project specific preferences for working with AI)
  • Co-pilot, not pilot
  • Tell the agent what tools you have available (CLI, MCP, etc.)
  • ALWAYS take the free review
  • Use as a learning tool
  • Don't trust anything blindly
  • Try staying in the IDE and avoiding external searches
  • Try working entirely in an agent terminal without opening the IDE
  • Before starting any task, experiment and see what AI can do with it
