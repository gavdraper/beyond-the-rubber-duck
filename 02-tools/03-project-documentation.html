Agent Behaviors

Define Agent Behavior

Describe how the AI should approach coding tasks:
  - Coding practices and standards to follow
  - Testing methodology and coverage expectations
  - Code review and quality assurance processes
  - Communication style and decision-making approach

Iterative Improvement

Each time you observe behavior you don't like:
  - Document the specific issue in your instructions
  - Add explicit guidance to prevent recurrence
  - Be specific about what you want instead
  - Test the updated instructions on new tasks

Example:
"ALWAYS ask before adding new dependencies"
instead of hoping the AI will ask naturally
