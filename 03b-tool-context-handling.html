Presenter Notes

  • GitHub Copilot: Uses ChatModes (Ask/Edit/Agent) with full context transfer
  • Serial Limitation: Mode switches carry complete conversation history
  • Claude Code Innovation: Subagents with isolated contexts enable parallelization
  • Context Pollution: Claude's approach reduces cross-task interference
  • Scalability: Parallel execution allows handling multiple simultaneous tasks
  • Trade-offs: Less context sharing vs. better performance and isolation
